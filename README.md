# Scratch Tools

一个方便个人日常使用的小工具仓库。

## 工具列表

### [Vault Architect - Deterministic Password Generator](./gen_password_scratch/)

一个安全、确定性的密码生成器，能够根据相同的输入在不同设备上生成相同的密码。

**在线使用：** https://tallyblue123.github.io/scratch_tools/gen_password_scratch/web/index.html

详细文档请查看 [gen_password_scratch/README.md](./gen_password_scratch/README.md)


### [Productivity Widgets](./productivity-widgets/)

一个 Windows 桌面透明小组件，集成月历与四象限待办事项（Eisenhower Matrix），支持重复任务和窗口置顶。基于 Tauri 构建。
